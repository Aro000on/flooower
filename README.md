# flooower - Авторская флористика и доставка цветов

![flooower](https://meta.iaron.dev/flooower/og.jpg)

Веб-ресурс, созданный в рамках выполнения практической работы по модулю **ПМ.04 «Создание медиафайлов для контента веб-ресурсов»**. Проект представляет собой современный интернет-магазин цветов, выполненный в нежных пастельных тонах с упором на качественный визуальный контент.

## 🌸 О проекте

**flooower** - это не просто сайт, а демонстрация полного цикла подготовки медиаконтента: от подбора исходных изображений до их глубокой обработки, оптимизации и интеграции в рабочую веб-среду.

### Основные возможности:
* **Адаптивный дизайн:** Сайт корректно отображается на смартфонах, планшетах и десктопах.
* **Интерактивная корзина:** Реализована на чистом JavaScript (добавление товаров, расчет суммы, сохранение состояния).
* **Плавная навигация:** Использование современных CSS-анимаций (fade-in) и плавного скролла к разделам.
* **SEO-подготовка:** Настроены мета-теги, Open Graph и канонические ссылки.

## 🖼️ Обработка медиафайлов (Техническая часть)

В соответствии с требованиями задания, весь графический контент прошел следующие этапы:

1.  **Кадрирование:** Все изображения товаров приведены к единому формату **1:1** для соблюдения визуальной иерархии.
2.  **Цветокоррекция:** Использование единого пресета в пастельных тонах для создания гармоничного стиля бренда.
3.  **Ретушь:** Устранение дефектов на лепестках, очистка фона и удаление визуального «шума».
4.  **Оптимизация:**
    * Основной формат: **WebP** (обеспечивает минимальный вес при высоком качестве).
    * Запасной формат: **JPEG**.

## 📂 Структура проекта

* `index.html` - главная страница магазина.
* `style.css` - стили проекта (включая адаптивную сетку и анимации).
* `script.js` - логика корзины и интерактивные элементы.
* `/img` - папка с медиафайлами:
    * `im1.webp` ... `im10.webp` - обработанные файлы.
    * `im1.jpg` ... `im10.jpg` - оригиналы.

## 🛠️ Инструменты
* **GIMP** - ретушь и цветокоррекция.
* **ILoveIMG** - финальная компрессия и конвертация в WebP.
* **VS Code** - верстка и программирование.
